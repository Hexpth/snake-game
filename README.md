# 🐍 Snake Game — Nokia 3310 Style

A classic Snake game built with vanilla HTML, CSS and JavaScript.
Designed to look like an old Nokia 3310 phone.

![Snake Game Preview](preview.png)

## 🎮 Live Demo

[Play the game](https://YOUR-NICK.github.io/snake-game/)

## 📁 Project Structure

```
snake-game/
├── index.html   — page structure
├── style.css    — phone styles
├── script.js    — game logic
└── README.md    — this file
```

## 🚀 How to Run

### Option 1 — Open locally
1. Download or clone the repository
2. Open `index.html` in your browser

### Option 2 — Clone via Git
```bash
git clone https://github.com/YOUR-NICK/snake-game.git
cd snake-game
open index.html
```

## 🕹️ Controls

| Key | Action |
|-----|--------|
| W | Move up |
| S | Move down |
| A | Move left |
| D | Move right |
| P | Pause / Resume |
| R | Restart |
| T | Table with results |

On mobile — use the **D-Pad** buttons on the phone screen.

## ⚙️ Features

- 📱 Nokia 3310 phone design
- 🟢 LCD green screen effect
- 🐍 Classic snake gameplay
- 🍎 Blinking food
- ⚡ Speed increases as you eat
- ⏸️ Pause and restart buttons
- 📱 Mobile friendly with touch controls
- 🎮 Keyboard support (Arrows / WASD)
- 🔢 Score counter in Nokia style (0000)

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API
- Google Fonts — Press Start 2P
